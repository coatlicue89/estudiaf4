# Clase 16 — Modelo de Bohr del hidrógeno, Wilson-Sommerfeld y postulado de de Broglie

## Repaso: postulados de Bohr (1913)

1. Órbita circular, atracción Coulombiana, leyes de la mecánica clásica.
2. Momento angular orbital cuantizado: $L = n\hbar$, con $\hbar = \dfrac{h}{2\pi}$.
3. Órbita permitida → sin radiación EM $\Rightarrow$ energía $E$ constante.
4. Se emite radiación EM en transiciones entre estados con frecuencia
   $\nu = \dfrac{E_i - E_f}{h}$.

## Consecuencias para el átomo de hidrógeno

### Postulado 1: órbita circular clásica

$$\frac{Z e^2}{r^2} = \frac{m v^2}{r} \quad \Rightarrow \quad Z e^2 = m r v^2$$

Fuerza central → $L = \text{cte} = m v r$.

### Postulado 2: $L = n\hbar$

$$m v r = n\hbar, \qquad n = 1, 2, 3, \dots \quad \Rightarrow \quad
v = \frac{n\hbar}{m r}$$

Combinando con el postulado 1:

$$Z e^2 = m r v^2 = m r \frac{n^2 \hbar^2}{m^2 r^2} = \frac{n^2 \hbar^2}{m r}$$

$$\Rightarrow \boxed{\,r_n = \frac{n^2 \hbar^2}{m Z e^2}\,}
\qquad n = 1, 2, 3, \dots \quad \longrightarrow \text{órbitas cuantizadas}$$

$$\Rightarrow v_n = \frac{Z e^2}{n\hbar}, \qquad n = 1, 2, 3, \dots$$

### Energía de las órbitas

$$T = \frac{1}{2} m v^2 = \frac{Z e^2}{2 r}, \qquad V = -\frac{Z e^2}{r}$$

$$E = T + V = -\frac{Z e^2}{2 r}$$

$$\Rightarrow E_n = -\frac{Z e^2}{2 n^2 \hbar^2}\, m Z e^2
= -\frac{m Z^2 e^4}{2 n^2 \hbar^2} = -\frac{E_H}{n^2}, \qquad n = 1, 2, 3, \dots$$

con $Z = 1 \Rightarrow E_H = 13{,}6\ \text{eV}$.

### Series espectrales

Las transiciones vistas en la espectroscopía tienen energías
$E_m - E_n = h\nu_{mn}$.

Si $n = 1$, $m > 1$:

$$h\nu_{m1} = -\frac{E_H}{m^2} + \frac{E_H}{1^2}
= E_H \left(\frac{1}{1^2} - \frac{1}{m^2}\right)
\qquad \text{Serie de Lyman (ultravioleta)}$$

Si $n = 2$, $m > 2$:

$$h\nu_{m2} = E_H \left(\frac{1}{2^2} - \frac{1}{m^2}\right)
\qquad \text{Serie de Balmer (visible)}$$

## Átomo de de Broglie - Bohr

$$L = m v r = n\hbar = \frac{n h}{2\pi}$$

$$\Rightarrow 2\pi r = \frac{n h}{m v} = \frac{n h}{p} = n\lambda$$

La circunferencia de la órbita contiene un número entero de longitudes de onda
de de Broglie (onda estacionaria; p. ej. $n = 3$).

## Wilson-Sommerfeld (1916)

Regla de cuantización de sistemas periódicos:

$$\oint p\, dq = n h$$

donde $q$ es una coordenada ($x$, $\phi$) y $p$ su momento conjugado
($p_x$, $L_z$).

Partícula en órbita circular con fuerza central:

$$\oint L\, d\phi = n h \;\Rightarrow\; L \oint d\phi = L\, 2\pi = n h
\;\Rightarrow\; L = n\hbar \quad \leftarrow \text{condición de Bohr}$$

## Repaso: fotones y luz

$$\text{ondas:}\ \lambda = \frac{v}{\nu}; \qquad \text{luz:}\ v = c
\;\Rightarrow\; \lambda = \frac{c}{\nu}$$

Einstein, cuantización del efecto fotoeléctrico: $h\nu = E$. Entonces
$\lambda = \dfrac{hc}{E}$, y con relatividad $E = c p$:

$$\lambda = \frac{h}{p} \;\Rightarrow\; p = \frac{h}{\lambda}\quad (\text{fotón})$$

La versión ondulatoria de la luz es la Ec. de Maxwell ⇒ ecuación de ondas
clásica:

$$\left(\frac{\partial^2}{\partial x^2} - \frac{1}{c^2}
\frac{\partial^2}{\partial t^2}\right) \psi(x, t) = 0
\qquad (\psi: \text{campo } E \text{ o } B)$$

Proponiendo solución onda plana $\psi \sim e^{i(\omega t - k x)}$:

$$-k^2 + \frac{\omega^2}{c^2} = 0 \;\Rightarrow\; \boxed{\omega = c k}
\qquad \text{relación de dispersión}$$

$$\text{velocidad de fase:}\ v_f = \frac{\omega}{k} = c; \qquad
\text{velocidad de grupo:}\ v_g = \frac{d\omega}{dk} = c$$

Como la velocidad de fase es independiente de $\omega$, la EO clásica es
**no-dispersiva**: los paquetes de onda no se deforman.

## Postulado de de Broglie

Partícula de impulso $p$ y energía total (relativista) $E$ tiene asociada una
onda con:

$$\lambda = \frac{h}{p}, \quad \lambda = \frac{1}{k} \;\Rightarrow\;
k = \frac{p}{h} \qquad (1)$$
$$\nu = \frac{E}{h} \qquad (2)$$

### Relación de dispersión $\nu = \nu(k)$

Dato: energía relativista $E^2 = c^2 p^2 + m_0^2 c^4 \Rightarrow
E = \sqrt{c^2 p^2 + m_0^2 c^4}$.

Usando (1) y (2): $h\nu = \sqrt{c^2 h^2 k^2 + m_0^2 c^4}$, entonces

$$\nu = \sqrt{c^2 k^2 + \frac{m_0^2 c^4}{h^2}}
= \sqrt{c^2 k^2 + \nu_0^2}$$

$$\boxed{\,\nu(k) = \sqrt{c^2 k^2 + \nu_0^2}\,}
\qquad \text{con } \nu_0^2 = \frac{m_0^2 c^4}{h^2}$$

### Velocidades

$$v_f = \frac{\nu}{k} = \frac{1}{k}\sqrt{c^2 k^2 + \nu_0^2}
= \sqrt{c^2 + \frac{\nu_0^2}{k^2}} > c$$

$$v_g = \frac{d\nu}{dk} = \frac{c^2 k}{\sqrt{c^2 k^2 + \nu_0^2}}
= c^2 \frac{p}{E} = c^2 \frac{m v}{m c^2} = v < c$$

Notar que $v_f\, v_g = c^2$.

## Ondas piloto

De Broglie nos dice: el estado de partícula **libre** es una onda

$$\psi(x) \sim e^{-i 2\pi [k x - \nu(k) t]}$$

con relación de dispersión $\nu(k) = \sqrt{c^2 k^2 + \nu_0^2}$.

### Principio de superposición

Se pueden superponer muchas "ondas piloto" para describir estados más complejos,
pero como $p = \hbar k$, ya no tienen bien definido el momento.

### Deslocalización

Notar que la onda anterior tiene totalmente indefinida la posición $x$: la
partícula puede estar en cualquier lado.

## Paquetes de onda

$$\psi(x, t) = \int_{-\infty}^\infty dk'\, A(k')\,
e^{i 2\pi (k' x - \nu t) - i\varphi(k')}$$

donde $A(k')$ es la distribución espectral.

En general, las partículas tienen una cierta localización, como la luz que
aparece en "trenes de onda" cuando es emitida por un átomo, con un
$\Delta t \sim 10^{-9}$ s y longitud de coherencia.
